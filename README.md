# Barra Connect Site

Site estatico em HTML + Tailwind CDN, com seções divididas em parciais carregadas via JavaScript.

## Estrutura
- `index.html`
- `partials/`
- `css/`
- `js/`
- `assets/`

## Como visualizar
Abra via servidor local (ex: Live Server). `fetch` de parciais pode bloquear em `file://`.

## Deploy
Pronto para deploy em Vercel/Netlify como site estatico.
