Site estático da Barra Connect com páginas em HTML, estilos via Tailwind CDN e seções divididas em parciais carregadas via JavaScript.

**Site:** [barra-connect-site.vercel.app](https://barra-connect-site.vercel.app/)

## Páginas
- `index.html` (home)
- `quem-somos.html` (institucional)

## Funcionalidades
- Seções em parciais (`partials/`) carregadas dinamicamente
- Cadastro rápido com envio automático para WhatsApp
- Layout responsivo (mobile e desktop)

## Stack
- HTML + Tailwind CSS (CDN)
- JavaScript (vanilla)
- Ícones Lucide

## Estrutura
- `index.html`
- `quem-somos.html`
- `partials/`
- `css/`
- `js/`
- `assets/`


## Onde editar
- Conteúdo e seções: `partials/*.html`
- Lógica de WhatsApp e formulário: `js/app.js`
- Cores principais: `css/styles.css` e `js/tailwind-config.js`
- Imagens e logos: `assets/`
